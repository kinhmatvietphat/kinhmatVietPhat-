# Product QR Web Demo

Web quản lý sản phẩm + tạo QR code + xem sản phẩm khi quét QR.

## Hướng dẫn

1. Tạo Firebase Project, bật Firestore + Storage
2. Lấy firebaseConfig → thay vào `admin.js` và `product.js`
3. Chạy local: mở `index.html` bằng Live Server
4. Deploy online: Render → Static Site → chọn repo
